Download

Metawidget downloads are hosted on SourceForge.

There are two downloads: one containing binaries, documentation and examples, and the other containing source code and tests.

Product Download Version Release Date  
Metawidget Binaries, Documentation and Examples ${metawidgetVersion} Download
Source Code and Tests ${metawidgetVersion} Download

A Maven repository is also available at http://repository.jboss.org/nexus/content/groups/public-jboss/org/metawidget/metawidget (thanks to JBoss for hosting). Add it to your pom.xml:

<dependency>    <groupId>org.metawidget</groupId>    <artifactId>metawidget</artifactId>    <version>${metawidgetVersion}</version> </dependency>

If you need Metawidget re-packaged in different forms, such as separate frontend and backend JARs, different Maven classifiers are also provided.
