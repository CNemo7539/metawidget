Download

Metawidget downloads are hosted on SourceForge.

There are two downloads: one containing binaries, documentation and examples, and the other containing source code and tests.

Product Download Version Release Date  
Metawidget Binaries, Documentation and Examples ${metawidgetVersion} Download

Maven

Metawidget distributions are also deployed at https://repository.jboss.org/nexus. Add the following repository definition to your pom.xml...

<repository>
   <id>jboss-public-repository-group</id>
   <name>JBoss Public Maven Repository Group</name>
   <url>https://repository.jboss.org/nexus/content/groups/public-jboss/</url>
   <layout>default</layout>
</repository>

...and the dependency configuration as:

<dependency>
   <groupId>org.metawidget</groupId>
   <artifactId>metawidget</artifactId>
   <version>${metawidgetVersion}</version>
</dependency>

If you need Metawidget re-packaged in different forms, such as separate frontend and backend JARs, different Maven classifiers are provided. Thanks to JBoss for hosting.
