Contributing

Metawidget is developed by the community. You can contribute by providing feedback on the forums, by reporting issues, by adding to the Wiki and by contributing enhancements to the code and documentation.

Full source code and documentation is included in the download. Some starting points in the documentation include:

  • Architecture overview
  • Implementing your own Metawidget
  • Implementing your own Inspector

You can also browse the source code online, such as source code for the supported front-ends and source code for the supported back-ends.

Some enhancements suggested by users to date include:

Suggestion Description
Maven support Metawidget integrates with lots of third-party libraries, each of which is continually being updated by their respective owners. Adding Maven support would help other contributors maintain version dependencies.
SWT support Metawidget's desktop support currently only includes Swing. A version for SWT, probably modelled closely off SwingMetawidget, would expand our desktop coverage.
ZK support ZK is a very popular Web framework, and has interesting architectual differences compared to our currently supported Web frameworks.
OSGi support OSGi is an important standard for modular development. Adding OSGi headers to the Metawidget JAR would make it more useful in many environments. See http://www.aqute.biz/Code/Bnd.
